Configuration du lien

Lien universel : Ce lien sera valide tant que le contrat de location est actif. Le locataire pourra l'utiliser pour tous ses paiements (loyer, caution, etc.). Le système détectera automatiquement le type de paiement selon le contexte.

Détection automatique

  • Caution : Si non payée, sera déduite automatiquement du montant versé
  • Loyer : Le reste du montant sera considéré comme paiement de loyer
  • Flexible : Le locataire peut saisir n'importe quel montant
